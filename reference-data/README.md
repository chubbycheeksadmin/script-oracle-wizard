# Reference Data

Drop real project files here to improve the platform's accuracy.

## Folder Structure
- **scripts/** — Production scripts (PDF or text)
- **budgets/** — Final budgets (Excel, CSV, PDF)
- **schedules/** — Shooting schedules (any format)
- **bid-letters/** — Bid letters (Word/PDF)

## What to include per project
At minimum: budget total + shoot days + type (studio/location/both)
Ideally: script + full budget + schedule + bid letter

## How to add a project
1. Drop files into the relevant folders
2. Add an entry to metadata.json with the details
3. Henry can help structure/index if needed
